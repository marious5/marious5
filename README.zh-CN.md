## 重点项目

### [Timer_Harmony](https://github.com/marious5/Timer_Harmony)

一个基于 HarmonyOS 的智能日程管理应用，使用 ArkTS 和 ArkUI 开发。

- 通过 AI 解析自然语言并创建日程
- 登录/注册流程与持久化登录态
- 日程增删改查、详情页、完成/归档流程
- 本地关系型存储与面向同步的数据层
- 根据待完成日程数量切换动态图标
- 面向 phone、tablet、2in1、wearable 的多设备配置

**技术栈：** HarmonyOS、ArkTS、ArkUI、Stage Model、RDB、HTTP API 对接

### [Cybar_Harmony](https://github.com/marious5/Cybar_Harmony)

一个面向鸡尾酒配方、推荐、自定义配方创建和社交互动的 HarmonyOS 客户端。

- 配方列表、详情、搜索、推荐、收藏和用户页面
- 点赞、收藏、评论和用户资料相关接口对接
- 自定义配方创建，包含配料选择、图片处理、酒精度计算和 AI 生成流程
- 深浅色主题支持与移动端优先的 ArkUI 页面
- 面向用户、配方和评论的管理页面

**技术栈：** HarmonyOS、ArkTS、ArkUI、REST APIs、本地缓存、图片选择器、主题管理
