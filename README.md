## Featured Projects

### [Timer_Harmony](https://github.com/marious5/Timer_Harmony)

An intelligent schedule management app for HarmonyOS, built with ArkTS and ArkUI.

- Natural-language schedule creation with AI parsing
- Login/register flow with persistent session handling
- Schedule CRUD, detail pages, completion/archive flow
- Local relational storage and sync-oriented data layer
- Dynamic app icon logic based on pending schedule count
- Multi-device target configuration for phone, tablet, 2-in-1, and wearable

**Tech:** HarmonyOS, ArkTS, ArkUI, Stage Model, RDB, HTTP API integration

### [Cybar_Harmony](https://github.com/marious5/Cybar_Harmony)

A HarmonyOS client for cocktail recipes, recommendations, custom recipe creation, and social interactions.

- Recipe list, detail, search, recommendations, favorites, and user pages
- Like, favorite, comment, and profile-related API integration
- Custom recipe creation with ingredient selection, image handling, ABV calculation, and AI generation flow
- Dark/light theme support and mobile-first ArkUI pages
- Admin-facing pages for users, recipes, and comments

**Tech:** HarmonyOS, ArkTS, ArkUI, REST APIs, local cache, image picker, theme management
